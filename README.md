# CA Intermediate Study Tracker

A Flask-based web application designed to help CA Intermediate students track study progress, monitor chapter completion, log study hours, and stay consistent with daily goals.

Live Demo:
https://ca-intermediate-study-tracker.onrender.com

## Features

- User Registration & Login
- Automatic ICAI Syllabus Seeding
- Study Session Logging
- Subject-wise Hour Tracking
- Daily Study Goals
- Study Streak Tracking
- Chapter Completion Tracking
- Subject Search
- Group 1 & Group 2 Progress Dashboard
- Dark Mode / Light Mode
- Study Analytics & Trend Charts

## Installation

```bash
git clone https://github.com/tiramisubees/CA-Intermediate-Study-Tracker.git
cd CA-Intermediate-Study-Tracker
python -m pip install -r requirements.txt
python run.py
```
